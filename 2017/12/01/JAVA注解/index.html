用途：

如果没有用来读取注解的方法和工作，那么注解也就不会比注释更有用处了。但是在web开发中使用可以定义注解然后通过aop或者扫描包（比如说spring中提供的功能）来实现通过注解定位，更灵活方便。使用注解的过程中，很重要的一部分就是创建于使用注解处理器。Java SE5扩展了反射机制的API，以帮助程序员快速的构造自定义注解处理器。
